Grav

Grav is a modern opEn source flat-file CMS

¿Qué es Grav?

GRAV es un sistema administrador de contenido CMS. GRAV es un sistema tal como lo es WordPress o Joomla, etc. sin embargo entre muchas otras ventajas comparativas con de Joomla y Wordpress, GRAV no almacena el contenido ni la configuración en Bases de Datos, todo es almacenado en archivo de texto.

Usabilidad e interface

Utiliza markdown para gestionar contenido. Simple y facil de usar.

Tecnología

PHP. Facil instalación solo pega los archivos en un servidor con los requerimientos mínimos y listo.

Valor

Es un proyecto de código abierto bajo la licencia MIT, de uso gratuito.

Features especiales

Responde casi tan rápido como páginas estáticas, utilizando su sistema de Cache Avanzado.

Sitio web

https://getgrav.org
Contentful

Run your content as a service

¿Qué es Contentful?

Contentful es una API para contenidos, basada en la nube y que sirve para distribuir los contenidos a cualquier dispositivo y aplicación utilizando APIs REStful.

¿Cómo funciona? El editor gestiona el contenido en una plataforma web con una interfaz muy sencilla y fácil de usar. Tus desarrolladores acceden a este contenido utilizando una API de entrega de contenidos.

Lo más importante de Contentful, es que quiere ser un CMS, Pero no un CMS cualquiera; Contentful es un CMS que se centra primero en la API. Esto significa que todo el contenido se accede por medio de una API.

Gracias a su enfoque centrado en el contenido, también te permite crear modelos de contenido personalizado, por si no te gustan los modelos que existen predefinidos. La idea es crear contenido, no plantillas. Se desarrolla posteriormente alrededor del contenido.
